## 🚀 使用方法
### 方式一：直接打开（推荐）
1. 找到文件 `paw_patrol_english.html`
2. 双击用浏览器打开（推荐 Chrome、Safari、Edge）
3. 点击"开始学习"按钮
4. 点击单词或表情听发音
### 方式二：在手机上使用
1. 将 HTML 文件发送到手机
2. 用手机浏览器打开
3. 添加到主屏幕（类似 APP）
### 方式三：部署到网络
1. 上传到任意 web 服务器
2. 分享链接给小朋友
3. 可在线学习

## ⚠️ 注意事项
### 1. 浏览器兼容性
✅ 完美支持：
- Chrome 70+
- Safari 12+
- Edge 79+
- Firefox 75+
❌ 不支持：
- IE 浏览器
- 老旧版本浏览器

- 首次播放需要用户交互（点击）
- 部分安卓手机需要调大媒体音量
