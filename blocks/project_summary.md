# 汪汪队主题英语学习小游戏
## 项目介绍
这是一个专为幼儿园小朋友设计的汪汪队主题英语启蒙小游戏，通过可爱的界面和互动问答帮助小朋友轻松学习基础英语词汇。
## 功能特点
- 🐶 **六个汪汪队角色**，每个角色对应不同主题词汇
- 📚 **共60个基础英语单词**，涵盖不同生活场景
- 🔊 **语音朗读功能**，标准美式英语发音
- 🎨 **活泼可爱的设计风格**，贴合小朋友审美
- 🎯 **选择题互动**，简单易操作，适合小朋友
- 🏆 **积分奖励系统**，答错不惩罚，鼓励学习
- 🎉 **满分彩带庆祝**，增强成就感
- 📱 **全响应式设计**，支持手机/平板/电脑

## 使用方法
### 直接使用
直接用浏览器打开 `index.html` 文件即可开始游戏，不需要任何服务器或安装依赖。
### 部署为小程序
如果需要部署为微信小程序，可以将HTML/CSS/JS转换为小程序原生代码，结构已经分离清晰，转换非常方便。

## 技术特点
- 纯HTML/CSS/JavaScript开发，无需任何框架
- 使用Web Speech API实现语音朗读
- CSS动画实现流畅交互效果
- Google Fonts提供可爱圆润的字体
- 响应式网格布局，适配各种屏幕尺寸
