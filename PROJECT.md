# 🐾 汪汪队英语乐园
> 一个专为幼儿园小朋友设计的英语学习小游戏
## 🎯 项目特点
- 🐾 汪汪队立大功主题
- 🔊 每个单词都有标准发音
- 📱 完美适配移动端
- 🎮 趣味互动学习
## 🚀 快速开始
1. 打开 `index.html` 即可开始学习
2. 点击单词听发音
3. 切换主题学习不同类别单词
## 📚 学习内容
- 96 个英语单词
- 8 大主题（动物、水果、颜色、数字等）
- 英语发音 + 中文解释
## 🛠️ 技术栈
- HTML5 + CSS3 + JavaScript
- Web Speech API（语音合成）
- 响应式设计
